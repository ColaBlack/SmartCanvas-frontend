# react-model

#### 介绍

react-model是一个基于我自己的SmartCanvas项目抽象的react项目模版，也是基于ant design pro进行的二次开发。

#### 技术栈

- 基于ant design pro进行二次开发
- 使用react开发
- 使用ant design ui组件库
- 使用umi.js/openapi快速生成api接口
- 使用typescript进行代码编写，提高代码质量

#### 安装教程

1. 克隆本仓库
2. 安装依赖
3. 配置后端地址
4. 启动项目
